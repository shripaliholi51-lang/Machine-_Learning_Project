#Diabetes Prediction using Logistic Regression

This project predicts whether a person is likely to have diabetes using
machine learning (Logistic Regression).

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Dataset
The dataset contains basic medical details such as glucose level,
blood pressure, BMI, and age.

## Model Used
Logistic Regression

## Objective
To understand how classification models work in healthcare data
and practice machine learning concepts
